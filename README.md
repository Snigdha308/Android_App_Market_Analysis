# Android_App_Market_Analysis

Objective:
Analyze Google Play Store data to understand app categories, ratings, installs, and market trends.

Tools Used:
-Python
-Pandas
-NumPy
-Matplotlib
-Google Colab

Project Tasks:
-Data Cleaning
-Category Analysis
-Rating Analysis
-Install Analysis
-Data Visualization

Conclusion:
This project analyzes Google Play Store data and provides insights into app popularity, ratings, and market trends using data analytics techniques.
